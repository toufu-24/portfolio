# Node x Node

ニューラルネットワークを自分で構築・学習させ、ステージ攻略を目指すゲームです。\
スキルツリーを解放して強力なパーツを手に入れ、より難しいステージに挑みます。\
TensorFlow.js を採用し、ブラウザ上で手軽にニューラルネットワークの構築と学習を体験できます。
東京農工大学 MCC Hackathon 2026 Spring のチーム A 作品です。
