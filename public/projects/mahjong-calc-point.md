# mahjong-calc-point

手牌の画像から牌を認識し、麻雀の点数計算をサポートする Web アプリケーションです。\
カメラでの撮影と画像のアップロードに対応し、認識結果を確認・修正してから計算できます。\
手作業で牌を入力する手間を減らし、対局中でも手軽に点数を確認できるようにしました。
